## C++中的位运算技巧

> Marco
>
> 计算机的存储是用二进制形式存储的，使用位运算可以直接操作数据，是最快最直接的运算。掌握基本的位运算技巧，可以帮助我们简化程序代码和优化时间。
>
> 阅读估计耗时：2h

### 1. 位运算符有哪些

| 符号 | 意义 | 例子                    |
| ---- | ---- | ----------------------- |
| &    | 与   | `0001 & 0011 ==> 0001 ` |
| ^    | 异或 | `0001 & 0011 ==> 0010 ` |
| \|   | 或   | `0001 & 0011 ==> 0011 ` |
|      | 取反 | `~0001 ==> 1110 `       |
| <<   | 左移 | `0001 << 2 ==> 0100`    |
| \>>  | 右移 | `0001 << 2 ==> 0000`    |

> PS: 有一处特例，C/C++中，右移运算是算术移位，右移补的是符号位数字。
>
> `0001 >> 2 `变为`0000`
>
> `1001 >> 2 `变为`1110`

### 2. 常见使用

#### 2.1 判断奇偶

```C++
bool oddJudge(int &a){
    if(a&1 == 0) return true; // is odd
    else return false; // is not odd, 
}
```

​		假设`a=127`，记为`0x0fff`，`0x0fff & 0x0001`不难得出当前一个数字数字末位为1时，结果为`0x0001`，不然则是`0x0000`。

#### 2.2 交换两数

```c++
void swap(int &a, int &b){
    a ^= b;
    b ^= a;
    a ^= b;
}
```

​		理解上述函数需要先明白异或运算的一些基本数学定理。

> 1. a^a=0
> 2. a^0=a
> 3. a^1
> 4. 异或运算满足交换定律

​		然后根据这些数学定理，将上面展开就可以得到

> * b的最终结果为：`b = b^a`，从第一项开始展开为 `b = b^(a^b) ==> b = 0^a=a`
> * a的最终结果为：`a = a^b`，从第一项开始展开为 `a = (a^b)^(b^(a^b)) ==> 0^b=b`

#### 2.3 变换符号

```c++
int singReversal(int &a){
    return ~a+1;
}
```

​		也是展开就很好理解，具体为什么可以看前面写的文章。

> * -11变11
>
>   1111 0101 \==取反\==> 0000 1010 \==加一\==> 0000 1011
>
> * 11变-11
>
>   0000 1011 \==取反\==> 1111 0100 \==加一\==> 1111 0101

#### 2.4 求绝对值

```c++
int myAbs(int a){
    int i = a>>31; //求符号位
    return i==0?a:(~a+1);
}
```

​		当然可以进一步优化为

```c++
int myAbs(int a){
    int i = a>>31; //求符号位
    return ((a^i) - i);
}
```

​		具体展开如下:

> 如果a是正数，则i=0
>
> ​	a^0-0 ==>a;
>
> 如果a是负数，则i=-1
>
> ​	a^1+1 ==>-a;
